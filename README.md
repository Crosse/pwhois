pwhois
======

    $ ./pwhois -h
    Usage: pwhois [options]
    
    Specific options:
        -v, --[no-]verbose               Run verbosely
        -o, --output TYPE                Output style.  One of: csv, tsv, table, list
        -a, --attributes x,y,z           Attributes to return.
        -h, --help                       Show this message
